PartyHub

Martin Briefender, Carla Dimmler

Projektidee

PartyHub ist eine Plattform für Partys und Events in der Nähe, auf der man Veranstaltungen entdecken, ansehen und verwalten kann.

Gemeinsame Basis

  • Backend, Daten
  • Partys anzeigen
  • Profile Ansicht
  • Authentifizierung über Keycloak

Web App / ITP Projekt

  • Partys suchen, filtern, sortieren
  • Partys erstellen, bearbeiten, löschen
  • Profile durchsuchen
  • Follower, Following verwalten
  • Direkt im Browser nutzbar
  • Responsive, Progressive Web App

iOS App / MC

  • AirDrop, Teilen
  • Apple Karten
  • Standortbasierte Aktionen
  • Time Tracking
  • Partys erstellen, bearbeiten und löschen
  • Navigation per Tabs
