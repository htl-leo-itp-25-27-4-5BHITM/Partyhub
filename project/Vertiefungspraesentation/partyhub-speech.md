# Sprechtext PartyHub

## 1. Titel

PartyHub ist ein Projekt für Partys und Events in der Nähe.

## 2. Projektidee

PartyHub ist eine Plattform für Partys und Events in der Nähe. Dort kann man Veranstaltungen entdecken, ansehen und verwalten.

## 3. Gemeinsame Basis

Beide Versionen nutzen dieselben Daten. Man kann Partys anzeigen, Profile ansehen und sich per Keycloak anmelden oder registrieren.

## 4. iOS App

Die iOS App ist die native Version.

Dort gibt es AirDrop, Apple Karten, standortbasierte Aktionen, Time Tracking, das Erstellen, Bearbeiten und Löschen von Partys und die Navigation per Tabs.

## 5. Web App / ITP Projekt

In der Web App kann man Partys suchen, filtern und sortieren, Partys erstellen, bearbeiten und löschen, Profile durchsuchen und Follower sowie Following verwalten.

Sie läuft direkt im Browser und ist auch als Progressive Web App umgesetzt.
