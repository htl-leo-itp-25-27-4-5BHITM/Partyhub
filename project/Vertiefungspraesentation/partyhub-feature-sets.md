# PartyHub Feature Sets

## Gemeinsame Basis

- Backend / Daten
- Partys anzeigen
- Party-Details öffnen
- Profile ansehen
- Benachrichtigungen nutzen
- Authentifizierung über Keycloak

## Web App

- Partys suchen, filtern und sortieren
- Partys erstellen / bearbeiten / löschen
- Benachrichtigungen verwalten
- Profile durchsuchen
- Direkt im Browser nutzbar
- Responsive, Progressiv Web App

## Swift App

* Native iPhone-App in SwiftUI
* AirDrop / Teilen

* Apple Karten
* Standortbasierte Aktionen

* Time Tracking
* Partys erstellen, bearbeiten und löschen

* Native Navigation mit Tabs
