## Run Locally

Clone the project

```bash
  git clone git@github.com:htl-leo-itp-25-27-4-5BHITM/Partyhub.git
```

Go to the project directory

```bash
  cd Partyhub
```

Setup for local development

```bash
  ./deploy-local.sh
```

Open the Website in your Browser

```http request
  http://localhost::8080
```
