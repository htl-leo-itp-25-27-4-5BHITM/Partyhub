
# PartyHub



## Run Locally

Clone the project

```bash
  git clone git@github.com:htl-leo-itp-25-27-4-5BHITM/Partyhub.git
```

Go to the project directory

```bash
  cd Partyhub
```

Start the server

```bash
  ./deploy.sh
```

Open the Website in your Browser

```bash
  http://localhost::8080
```
